The differences between Web 1.0, Web 2.0, Web 3.0:

  1. The interaction:
    - Web 1.0 was consider as read-only web with very little interaction, where consumer can exchange the information together but it was not possible to interact with the website.
    - Web 2.0 that is considered as read-write web, is also called participative social web, allows managing and assembling large global crowd with common interests in social interaction.
    - Web 3.0 is also known as the Semantic Web, targets on linked data sets, includes altering the Web into a database, web of document, Portable and Personal.
  2. The Advertising:
    - In Web 1.0 Companies publish content that people consume, Banner Advertising.
    - In Web 2.0 People publish content that other people can consume, companies build platforms that let people publish content for other people (e.g. Flickr, YouTube, Wikipedia, MySpace), Interactive Advertising.
    - In Web 3.0 People build applications that people can interact with, companies build platforms that let people publish services by leveraging the associations between people or special content (e.g. FaceBook, Google Maps), Behavioural Advertising.
  3. The main features:
    - Web 1.0: user could store, share, view and print digital pictures, Personal web sites,Content Management system.
    - Web 2.0: Podcasting, Blogging, Tagging, Curating with RSS, Social bookmarking, Social networking, Social media.
    - Web 3.0: Semantic Web, Artificial Intelligence, 3D Graphics, Connectivity, Ubiquity.
